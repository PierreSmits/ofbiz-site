Apache OFBiz® Community

There are many ways you can help make OFBiz better.
Download the code and try it out, browse the source code, study the documentation and help to improve it (several documents are in a public Wiki).
Search in the issue tracker for open issues or features that need to be implemented, take ownership of an issue and try fix it.
If you discover a bug or a missing feature, file it in the issue tracker or discuss it in the mailing lists.
Read and subscribe to the mailing list to get in touch with users and developers.

  • Issue Tracker (Jira)
