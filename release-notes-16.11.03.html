Release Notes for Apache OFBiz® 16.11.03

Apache OFBiz® is an open source product for the automation of enterprise processes that includes framework components and business applications.

Apache OFBiz 16.11.03, released in 2017-06-29, is the third release of the 16.11 series, that has been stabilized since November 2016.
