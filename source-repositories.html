OFBiz Source Repositories

home/Community/Source Repository

Browsing the code and contributions in each of the repositories

Anyone can clone and checkout the source code from our repositories on Github, via following links:

  • ofbiz-framework - the repository containing all base components
  • ofbiz-plugins - the repository containing various plugins
  • ofbiz-site - the repository containing all elements of our website
  • ofbiz-tools - the repository containing various tools to manage our products

Browsing the revisions in each of the repositories

You can browse revisions in the repositories via the Fisheye services of the ASF, using any of the following links.

  • ofbiz-framework
  • ofbiz-plugins
  • ofbiz-site
  • ofbiz-tools

Forking and cloning our repositories

Forking from our Github repositories and cloning is easily done via standard git commands and various Git applications, but also via built-in functions in IDEs like Apache Netbeans, Eclipse and IntelliJ IDEA.

To checkout the source code directly from our repositories, simply use the following command (if you are using a GUI client, configure it appropriately).

OFBIZ-FRAMEWORK

This repository contains all base functionalities of the OFBiz product. It also contains all release branches.

$ git clone http://github.com/ofbiz-framework.git

OFBIZ-PLUGINS

The ofbiz-plugins repository contains plugins maintained by us, and should preferably be cloned when you are within the 'ofbiz-framework' folder

$ git clone https://github.com/apache/ofbiz/ofbiz-plugins/trunk plugins

OFBIZ-SITE

This repository contains the pages shown on our website

$ git clone https://github.com/apache/site ofbiz-website

OFBIZ-TOOLS

This repository contains various project management tools

ofbiz tools
$ git clone https://github.com/apache/ofbiz/tools ofbiz-tools
